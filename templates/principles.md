## Pre-commit чеклист

Перед каждым коммитом проверь:

- [ ] Нет нарушений запретов из таблиц выше
- [ ] Сборка зелёная (компилируется без ошибок)
- [ ] Логи через именованные категории (не дефолтные)
- [ ] `follow_decl` где возможно
- [ ] Нет закомментированного кода
- [ ] Нет `TODO` / `FIXME` которые должны быть закрыты
- [ ] Публичный API не сломан (если применимо)
